# Better Lockpicking Continued

An unofficial maintenance continuation for Project Zomboid Build 42.19.

This continuation keeps the original mod's lockpicking features working with the current game build, including vehicle doors and trunks, building doors, windows, and hotwiring. It also provides optional per-feature settings through the game's native Mods menu.

## Status

Build 42.19 single-player functionality has been tested. Multiplayer support has static synchronization coverage but has not yet been play-tested.

## Installation

Subscribe through the Steam Workshop when the updated release is available, then enable **Better Lockpicking Continued** in the Mods menu for the save or server you want to use.

## Development

The mod targets Project Zomboid Build 42.19. Native settings do not require an additional Workshop dependency. When changing Lua code, restart the game completely and test with a fresh save before publishing.
