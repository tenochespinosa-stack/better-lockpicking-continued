# Better Lockpicking Continued

A continuation of a lockpicking mod for Project Zomboid Build 42.19.

It covers vehicle doors and trunks, building doors, windows, and hotwiring. Each feature can be enabled or disabled through the in-game Mods settings.

## Status

Tested in single-player on Build 42.19. Multiplayer synchronization is covered by static checks but has not been play-tested.

## Download

Get the latest version from [Releases](https://github.com/tenochespinosa-stack/better-lockpicking-continued/releases).

## Installation

Install through the Steam Workshop when available, or download the release ZIP and place the `BetterLockpickingContinued` folder in your Project Zomboid `mods` folder. Enable it for the save or server you want to use.
